# Create a SPADE policy for self deal proposal

```
NAME:
   singularity deal spade-policy create - Create a SPADE policy for self deal proposal

USAGE:
   singularity deal spade-policy create [command options] DATASET_NAME [...PROVIDER_ID]

OPTIONS:
   --min-delay value     Minimum delay in days for the deal start epoch (default: 3)
   --max-delay value     Maximum delay in days for the deal start epoch (default: 3)
   --min-duration value  Minimum duration in days for the deal start epoch (default: 535)
   --max-duration value  Maximum duration in days for the deal start epoch (default: 535)
   --verified            Whether to propose dea as verified (default: true)
   --price value         The price of the deal measured by per 32GiB over the whole duration (default: 0)
   --help, -h            show help
```

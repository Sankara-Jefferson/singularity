# Data Source

```sh
NAME:
   singularity datasource - Data source management

USAGE:
   singularity datasource command [command options] [arguments...]

COMMANDS:
   add      Add a new data source to the dataset
   list     List all sources
   remove   Remove a data source
   check    Check a data source by listing its entries
   update   Update the config options of a source
   rescan   Rescan a data source
   status   Get preparation status of a data source
   help, h  Shows a list of commands or help for one command

OPTIONS:
   --help, -h  show help

```

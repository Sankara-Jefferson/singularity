# 从导出的私钥导入钱包

{% code fullWidth="true" %}
```
命令名称：
   singularity wallet import - 从导出的私钥导入钱包

用法：
   singularity wallet import [命令选项] 私钥

选项：
   --help, -h  帮助信息

   Lotus

   --lotus-api value    Lotus RPC API端点 (默认值："https://api.node.glif.io/rpc/v1")
   --lotus-token value  Lotus RPC API令牌
```

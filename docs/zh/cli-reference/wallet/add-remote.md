# 添加远程钱包

{% code fullWidth="true" %}
```
名称：
   singularity wallet add-remote - 添加远程钱包

用法：
   singularity wallet add-remote [命令选项] <address> <remote_peer>

选项：
   --help, -h  显示帮助

   Lotus

   --lotus-api value    Lotus RPC API 端点 (默认值: "https://api.node.glif.io/rpc/v1")
   --lotus-token value  Lotus RPC API 令牌

```

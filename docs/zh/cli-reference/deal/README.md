# 复制 / 交易管理

{% code fullWidth="true" %}
```
名称:
   singularity deal - 复制 / 交易管理

用法:
   singularity deal 命令 [命令选项] [参数...]

命令:
   schedule      安排交易
   spade-policy  管理 SPADE 政策
   send-manual   发送手动交易提议到增量或遗留市场
   list          列出所有交易
   help, h       显示命令列表或一个命令的帮助

选项:
   --help, -h  显示帮助信息
```

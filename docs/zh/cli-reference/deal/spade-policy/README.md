# 管理 SPADE 策略

{% code fullWidth="true" %}
```
名称：
   singularity deal spade-policy - 管理 SPADE 策略

用法：
   singularity deal spade-policy 命令 [命令选项] [参数...]

命令：
   create   为自我交易提议创建一个 SPADE 策略
   list     列出所有自我交易提议的 SPADE 策略
   remove   删除一个自我交易提议的 SPADE 策略
   help, h  显示命令列表或某个命令的帮助

选项：
   --help，-h  显示帮助
```

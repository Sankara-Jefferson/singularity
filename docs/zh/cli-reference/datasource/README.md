# 数据源管理

{% code fullWidth="true" %}
```
名称：
   singularity datasource - 数据源管理

用法：
   singularity datasource 命令 [命令选项] [参数...]

命令：
   add      向数据集中添加新的数据源
   list     列出所有数据源
   status   获取数据源的数据准备概要
   remove   删除数据源
   check    通过列出其条目检查数据源。这并非列出已准备好的项目，而是直接列出数据源以验证数据源连接。
   update   更新源的配置选项
   rescan   重新扫描数据源
   daggen   生成并导出代表数据源完整文件夹结构的DAG
   inspect  获取数据源的准备状态
   help, h  显示命令列表或帮助某个命令

选项：
   --help, -h  显示帮助信息
```
